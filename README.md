## Repository Structure

```
notes_ba4/
├── _overview/          # Compiled PDF overviews for each subject
│   ├── iml/
│   ├── algo/
│   └── sigproc/
├── _shared/            # Shared LaTeX style files
│   ├── cheatsheet.sty
│   └── course.sty
├── iml/
├── algo/
└── sigproc/
```

Each subject folder contains `cheatsheet.tex` (the cheat sheet) and `course.tex` (extended notes).
